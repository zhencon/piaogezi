票格子，发票票据转excel工具。pdf、ofd、xml格式。发票批量改名、发票合并、入账标识、在线发票查验。
功能特点：
■ 支持win7-64位、win10-64位、win11-64位系统。不支持32位。
■ 无需手机注册，打开即用。
■ 无需上传文件，本地识别。
■ 内置OFD阅读器和XML阅读器。
■ 搭载PDF和OFD双引擎。
■ 电子发票批量转Excel。
■ 支持发票文件去重。
■ 支持发票金额统计。
■ 支持按发票号码进行文件改名。（一个文件多张发票不改名，压缩文件里的不改名）
■ 支持生成税务局入账导入模版，一键入账查验。
■ 支持发票全部的字段。显示在Excel第1个sheet工作表
■ 支持商品明细。显示在Excel第2个sheet工作表。
■ 支持商品税收编码提取。
■ 支持单页最大100行商品明细。
■ 支持单独商品清单。
■ 支持商品明细项中的商品名和规格型号换行合并。
■ 支持"全电发票"试点地区电子普通发票和电子专用发票。
■ 支持新版火车票PDF、OFD、XML格式。
■ 支持航空电子行程单PDF、OFD、XML格式。
■ 支持非税收入票据识别。
■ 单个可执行文件。
■ Portable便携式设计，可以拷贝到U盘随使用。
■ 绿色免安装，解压到文件夹后直接运行。
■ 免费版：未注册支持每次打开处理5个文件，每天最多使用3次。
■ 限制PDF文件最大50M。ZIP、RAR文件最大50M。
